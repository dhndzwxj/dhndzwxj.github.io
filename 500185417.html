如何计算31省区市的实际GDP增长率？

发表于2021-03-19|更新于2021-10-22|经济学数据
|字数总计:926|阅读时长:3分钟|阅读量:

笔者最近发现，用不变价格（全国）GDP和名义价格（全国）GDP的比值作为权重，乘以31省区市的GDP数据进行调整，想通过这种方法得到省一级的GDP实际增速数据。但把得到的结果和官方公布的相比发现差距非常之大。苦思冥想不得其解，在人大经管之家发现一个帖子分享的做法，照着做了，问题是解决了，但是个中原因我还未能探明。

计算各省的实际增速

第一步，选好基年，计算考察年份的基期地区生产总值指数

【需要的数据】地区生产总值指数。一般在《中国统计年鉴》“国民经济核算”章的“地区生产总值”节的最后几列。

计算方法：

本年的基期指标=上一年的基期指数×本年指数（上年=100）100\text{本年的基期指标}=\frac{\text{上一年的基期指数}\times\text{本年指数（上年=100）}}{100} 本年的基期指标=100上一年的基期指数×本年指数（上年=100）​

第二步，根据上一步求出来的基期地区生产总值指数，计算实际GDP

【需要的数据】第一步的基期地区生产总值指数，名义GDP（注意名义GDP只需要基期一年的！）

计算方法：

实际GDP=基年名义GDP×本年基期GDP指数100\text{实际GDP}=\frac{\text{基年名义GDP}\times\text{本年基期GDP指数}}{100} 实际GDP=100基年名义GDP×本年基期GDP指数​

Excel示例代码：

=($B5*B24)/100

第三步，基于实际GDP计算实际增长率

【需要的数据】各年实际GDP

这一步的算法十分简单：

GDP实际增长率=本期实际GDP−上期实际GDP上期实际GDP\text{GDP实际增长率}=\frac{\text{本期实际GDP}-\text{上期实际GDP}}{\text{上期实际GDP}} GDP实际增长率=上期实际GDP本期实际GDP−上期实际GDP​

处理数据时使用的Matlab代码：

a = unnamed;
r = zeros(32,23);
r(1,2:23) = (1998:2019)';
for year=2019:-1:1998
   i = 2020-year; 
   r(2:32,24-i) = a(31*i-30:31*i,1);
end

计算南北方实际增速

这次将单个省份的GDP平减指数加权加总得到南方和北方作为一个整体的GDP平减指数。计算公式如下：

北方GDP平减指数=∑i北方省份i名义GDP∑i北方省份i名义GDP北方省份i名义GDP平减指数\text{北方GDP平减指数} = \frac{\sum_i \text{北方省份}i\text{名义GDP}}{\sum_i\frac{\text{北方省份}i\text{名义GDP}}{\text{北方省份}i\text{名义GDP平减指数}}} 北方GDP平减指数=∑i​北方省份i名义GDP平减指数北方省份i名义GDP​∑i​北方省份i名义GDP​

【需要的数据】北方（南方）各省区市的名义GDP，北方（南方）各省区市的GDP平减指数。

运算的MatLab代码如下：

deflator = unnamed;
no_gdp = unnamed1;
result = [];
index_n = [1,2,3,4,5,6,7,8,15,16,27,28,29,30,31];%北方标号
%index_s = [];%南方标号
for year = 1998:2020
    i = year-1997; %循环指标
    d = deflator(:,i);
    g = no_gdp(:,i);
    north_g = 0;
    north_d = 0;%分母
    south_g = 0;
    south_d = 0;%分母
    nation_g = 0;
    nation_d = 0;%分母
    for province = 1:31
        nation_g = nation_g + g(province);
        nation_d = nation_d + g(province)/d(province);
        if ismember(province,index_n) %判断province是否为数组index_n的元素
            north_g = north_g + g(province);
            north_d = north_d + g(province)/d(province);
        else
            south_g = south_g + g(province);
            south_d = south_d + g(province)/d(province);
        end
    end
    deflator_north = north_g/north_d;
    deflator_south = south_g/south_d;
    deflator_nation = nation_g/nation_d;
    result = [result;year,deflator_north,deflator_south,deflator_nation];
end
result = ["年份","北方","南方","全国";result];

得到这个结果后，再重复“计算各省的实际增速”部分的过程即可。
