黄金时代的幻象5

吴越|发表于2023-04-02|更新于2023-04-02|读书笔记黄金时代的幻象
|字数总计:2.7k|阅读时长:8分钟|阅读量:

本书前三章从经济历史和地理的角度，描述了战后资本主义黄金时代的增长与动态变迁，并对相关的六大理论进行了分析评述。问题在于，尽管长期以来学界在探讨工资的增长、需求的下降、积累速率的放缓和技术变革、生产率的增长的关系上投入了大量的努力，但是并不存在一个表述这些联系的宏观动态数理模型。因此，仅凭借经济历史的复杂证据，难以区别不同的战后增长与变迁理论所导出的解释。

本书的主要工作，就是构建用以分析资本主义宏观动态的形式理论工具，并使用它们从经济历史中提取可用的证据。本书展示了其中的三种工具：一个基于生产率变化的需求和盈利能力正式理论；一个正式的技术变革理论；以及一个关于全球化的理论。在构建这三种理论工具后，作者按顺序展示了它们所提供的关于盈利能力、技术变革以及新兴工业化国家经济增长的证据。

第四章《劳动价值论和预期价格》主要介绍作为本书理论基石的一系列概念，并给出了正式的数理表示。这包括：资本主义生产、社会必要劳动、熟练劳动、异质劳动、生产性劳动、联合生产、固定资本、价值的衡量、生产周期、货币形式的利润率等问题。这些基础性概念和问题已被大量的文献探讨过，本书只是简要确定了概念的内涵和回顾了必要的理论历史，并没有进行深入的研究。作者定义的这些概念的内涵也大都符合传统方法。因此，略过此章，顺着作者构建的具体理论工具的发展，在必要时再回到此章寻找最基础的理论定义，是较为合适的读书笔记形式。

本次读书笔记主要介绍第五章《积累、技术变革和利润率趋于下降的规律》。第五章第一节简要回顾了古典经济学中经济增长理论的起源。第二节概括了马克思的平衡扩大再生产模型，并且从马克思主义传统中提炼出了黄金时代的增长模型。第三节研究危机理论，考察消费不足、利润挤压和趋于下降的利润率对于积累的限制。第四节中，作者明确地抛弃了均衡的传统假设，提出了资本主义经济的一个新模型，并考察了作为新模型的各种特殊形式的、关于经济增长和趋于下降的利润率的过往叙述。这一动态非均衡模型表明，可行的技术变革降低了整体利润率，资本主义绝不可能保持无间断的增长。增长的限制就深植于资本主义经济的基本运行机制之中。这一模型也为本书作者的核心论点提供了分析的理论基础：二十世纪七八十年代增长放缓是由内在原因导致的，而不是由和稳定性条件联系在一起的外在原因导致的。

第五章、积累、技术变革与趋于下降的利润率

（一）经济增长理论的古典起源

经济增长理论起源于重商主义者的著作。但是，直到重农主义者的出现，根植于生产关系的增长理论体系才终于形成。从亚当·斯密到约翰·穆勒，古典经济学家们都追随着重农主义者的理论先导，尽管他们抛弃了剩余产品只是在农业生产中才存在的自然现象这一观点。对于古典经济学家而言，增长来自于社会剩余的生产性投资。因此，剩余的来源、所有权和控制权以及用途，都需要理论解释。这一基本观点使得价值、收入分配和增长这些问题成为了一个不可分割的整体。利润率问题在早期的讨论中占据了突出的位置，它同时作为一种收入、一个投资基金的来源、对进一步投资的激励在不同理论中存在。

无论古典经济学家们的理论间存在何种区别，他们和与他们的同时代人都认为，通过一部分社会产品（利润）的投资实现的资本积累是经济的增长引擎。同时，斯密和李嘉图都提出了经济增长会达到静止状态的观点，但是他们为此提供了不同的解释。斯密认为利润因而进一步投资的激励会被竞争所侵蚀，李嘉图则主张递减的收益率最终将导致经济的停止。古典经济学家所提出的增长和危机的问题，最终被留给了马克思来解决。

（二）马克思的积累理论和危机理论

对马克思而言，经济增长或说资本积累是资本主义生产关系的扩大。由于篇幅的限制，这里不再详细介绍马克思的积累理论。仅记录提炼的作者马克思扩大再生产图式的核心框架，以及基于马克思扩大再生产图式的黄金时代增长路径的分析。

1 马克思的扩大再生产图示

马克思主义的扩大再生产模型是以两部门图式构成的：部门1生产生产资料，部门二生产生活资料。生产技术被认为保持不变。以下的积累图示都是以价值形式表示的：

\begin{eqnarray} &&C_{1t} + V_{1t} + S_{1t} = C_{1t+1} + C_{2t+1} \tag{5.1}\\ &&C_{2t} + V_{2t} + S_{2t} = V_{1t+1} + (1-b_t)(S_{1t}+S_{2t}) \tag{5.2} \end{eqnarray}

其中，1-b表示被资本家非生产性地消费掉的剩余价值所占的比例。

马克思的积累模型(5.1)-(5.2)有以下几个重要特征：

  • 不存在固定资本。所有的资本都在一个生产周期内被消费掉了。
  • 在所有部门的生产中，资本的周转率被认为是相同的。
  • 马克思假设，一直存在充足的劳动力来满足经济的需求，不论劳动的增长率和对劳动力的需求如何。一个等价的假设是，实际工资和劳动力的价值保持恒定。
  • 马克思的积累模型对于投资行为特征的假设是很特殊的。他认为，增长的速度是由生产资料生产部门，也就是部门1的资本家的行为决定的。部门1的资本家消费他们所攫取的剩余价值中的一个固定比例，并将其余投入积累。部门2的资本家的投资，是由部门1生产出的资本品投入的可得性所决定的。森岛通夫（1973）和德赛（1979）证明，这一假设意味着两部门的增长速度在一个生产周期后就会收敛。

2 黄金时代的增长路径

作者给出了一个实物投入产出形式的积累模型。这一模型保留了马克思主义再生产图示的基本框架，但是这一模型更为实际：

资本可以追寻利润率的差异在不同部门间流动；部门1并不能主导经济整体的增长速率；实际工资可以随着劳动力市场的状况而变化。没有一个单独的部门专门为经济体生产固定资本投入，经济各部门的实际工资也被认为是相等的。

\begin{eqnarray} &&x_{1t} = a_{11}(1+g_{1t})x_{1t} + a_{12}(1+g_{2t})x_{2t} \tag{5.3}\\ &&x_{2t} = D_{t+1}L_{t+1} + F_t \tag{5.4} \end{eqnarray}

其中

\begin{eqnarray} &&L_{1t} = l_1(1+g_{1t})x_{1t} + l_2(1+g_{2t})x_{2t} \notag\\ &&F_t = (1-b_t)(1-\lambda_2 D_t)L_t \notag \end{eqnarray}
  • $ a_{1j} $ 表示部门j的加总的资本投入系数*(aggregate capital input coefficient)*；
  • $ \lambda_1 $ 和 $ \lambda_2 $ 分别代表加总的资本品和消费品的价值；
  • D表示实际工资，这在两部门间是相等的；
  • L表示被雇佣的总劳动数量；
  • F表示资本家进行的非生产性消费
  • $ g_i $ 表示部门i的增长率

式（5.3）和(5.4)表明，经济体中两种商品的流动必须满足供给和需求相等。如果不等，生产过剩或生产不足就将发生，积累的过程就会中断。

式(5.3)表明，要保持平衡增长，两部门需要保持以下相对规模：

\begin{eqnarray} &&x_{1t} = \frac{a_{12}(1+g_{2t})x_{2t}}{1-a_{11}(1+g_{1t})} = Wx_{2t} \tag{5.5} \end{eqnarray}

一旦部门2的产出固定了，互补性的资本品的产出就被两部门的资本投入系数和增长率确定了。增长率衡量了社会生产的投入品的增加。经济整体的增长率是由平均利润率（利润率是指一个周转周期内的利润率，两部门的周转时间被认为是相等并恒定的）和利润被用于扩大再生产的比例决定的：

\begin{eqnarray} g_t = b_t\pi_t \notag \end{eqnarray}

部门利润率为：

\begin{eqnarray} \pi_{jt} = \frac{(1-\lambda_2 D_t)l_jx_{jt}}{\lambda_1a_1x_{jt}+\lambda_2D_tl_jx_{jt}} \notag \end{eqnarray}

经济整体的利润率是部门利润率的加权平均

\begin{eqnarray} &&\pi_t = \frac{(1-\lambda_2 D_t)L_t}{\lambda_1K_t+\lambda_2D_tL_t} \notag\\ &&K_t = a_{11}x_{1t} + a_{12}x_{2t} \notag \end{eqnarray}

在此模型中，经济体存在两种调节机制：

第一，实际工资的变化被认为是就业率的单调递增且连续的函数。

\begin{eqnarray} \frac{dD_t}{D_t} = f[d(\frac{L_t}{N_t})],\quad f^\prime>0 \notag \end{eqnarray}

其中 $ N_t $ 表示可雇佣的劳动力的规模。

第二，经济整体在每个周转周期内的新增投资，是利润总额和利润投入扩大再生产比率的乘积。

\begin{eqnarray} l_t = b_t(1-\lambda D_t)L_t \notag \end{eqnarray}

新增投资在每个部门分配的比例取决于两部门各自的利润率：

u_{1t}=\left\{ \begin{eqnarray} && R(\frac{\pi_{1t}}{\pi_{2t}}),\quad \pi_{1t},\pi_{2t}>0 \notag\\ && 1, \quad\pi_{1t}>0,\pi_{2t}<0\notag\\ && 0, \quad \pi_{1t}<0,\pi_{2t}>0 \notag \end{eqnarray} \right.

$ u_{2t} = 1-u_{1t} $ ， R是两部门利润率之比的单调递增且连续的函数。
