DSGE模型中的投入产出交互作用（翻译稿）

揭晓|发表于2020-12-21|更新于2023-03-07|计量与模型模型DSGE
|字数总计:4.1k|阅读时长:16分钟|阅读量:

Bisová, S., Javorská, E., Vltavská, K., Zouhar, J. (2013). Input-output Interactions in a DSGE Framework. Conference Paper. Available at: https://www.researchgate.net/publication/268208686_Input-output_interactions_in_a_DSGE_framework

摘要

尽管很多人宣传DSGE（动态随机一般均衡模型）具有微观基础，通常还是使用国家一级的综合数据进行经济分析。然而，欧盟成员国的国家统计局创建的国民核算中包含有关个别行业的详细信息，如投入产出表。在这篇论文中，我们提出了一个典型RBC模型的多部门扩展的初步研究，该模型允许使用多个部门的投入产出结构的数据。我们建立了一个简单的基准模型，该模型允许任意数量的部门具有任意的投入产出结构。在实际中使用中，该模型遇到的困难在于需要找到变量的近似稳态值。一般情况下，求解稳态问题是困难的；然而，我们确实提供了一个特殊的对称情况的解析解，它有时也可以为非对称问题提供一个足够接近的近似值。

1. 引言

在今天的宏观经济分析中，通常使用向量自回归（VAR）、向量误差修正模型（VEC）或DSGE模型，其中DSGE模型是这三种方法中最新的一种，并且越来越受到研究人员和监管机构的欢迎。例如，捷克国家银行开发了一个DSGE模型“g3”[1]，用于解释当前的经济现象和进行经济预测。使DSGE模型如此有利的一个关键特征是，它们建立在微观经济基础上，因此应该抵制卢卡斯的批评。然而，在我们看来，DSGE模型微观基础的想法与通常用于模型校准的数据的总体性质不符；国家层面的数据几乎总是被使用，即使所有欧盟成员国的国民账户中都有行业层面的数据，并在投入产出表中列示[2]。

必须指出的是，一些现有模型实际上确实将企业部门划分为少数几个“行业”，通常区分制造商和零售商。或如捷克共和国财政部开发和使用的休伯特模型（见[3]，[4]）那样，在制造商、零售商和进口商之间进行划分。然而，在这种情况下，（i）模型包含极少数行业，（ii）各个行业之间的区别并不遵循国民账户的结构，以及（iii）投入产出结构仅限于零售商和制造商之间的单边关系。本文的目的是建立一个简单的基线模型，允许任意数量的扇区和任意的输入输出结构。

据我们所知，目前仅有的开发类似模型的研究是Bouakez等人的工作[5]，[6]。尽管我们无疑受到了这些作品的启发，但我们的模型与Boukaez等人的模型在几个方面有所不同。首先，他们的模型是为了利用美国国民账户的结构而建立的——这与捷克共和国和欧盟作为一个整体的结构都不同；最重要的是，美国国民账户收集各个部门之间资本流动的数据（资本流量表），它允许将资本形成过程简单地纳入DSGE模型。当我们打算将来将我们的模型应用于捷克数据时，我们不得不采用一种不同的、间接的方法，将投入产出过程和资本存量创造结合起来，这证明了均衡推导的复杂性。其次，虽然Bouakez等人的模型采用了新凯恩斯主义的观点，但我们的模型基本上符合真实商业周期（RBC）文献。

2. 模型

模型经济由一个无限期居住的家庭和J个公司组成，每个企业代表一个部门。家庭是所有企业的唯一劳动力供给者，企业的生产既可以被家庭消费，也可以作为其他部门的物质投入，或者以资本投资的形式使用。在我们的处理中，我们使用了社会规划者模型，它应该给出与竞争均衡公式相同的均衡条件，但是符号更加简洁，见[7]。社会规划师的目标是将家庭的（贴现）效用最大化，即最大化

关于替代模型和校准的评论。

  • 在校准方面，μij\mu_{ij}μij​参数应直接取自投入产出表，jjj参数将从消费者价格指数中进行校准。此外，我们需要Cobb-Douglas生产函数的参数；我们打算使用基于指数法的估计。参数可以估计为当前价格中的总增加值与员工薪酬的比率。这些数据发表在国民账户中。

3. 模型求解

3.1 均衡条件

首先我们构造拉格朗日函数：

\begin{eqnarray} \mathscr{L}&=&\mathbb{E} \sum_{t=0}^{\infty} \beta^{t}\left[\log C_{t}-\phi \log H_{t}-\sum_{j} \lambda_{j t}\left(-Y_{j t}+C_{j t}+I_{j t}+\sum_{k} M_{j k t}\right)\right] \notag\\ &=& \mathbb{E} \sum_{t=0}^{\infty} \beta^{t}\left[\log C_{t}-\phi \log H_{t}+\sum_{j} \lambda_{j t}\left(-Y_{j t}+C_{j t}+K_{j t}-(1-\delta) K_{j, t-1}\right)-\sum_{i} \sum_{j} \lambda_{i t} M_{i j t}\right] \notag \end{eqnarray}

一阶条件是通过设置拉格朗日的偏导数（w.r.t.决策变量和拉格朗日乘数）为零得到的。在消除拉格朗日乘数和一些明显的代数操作后，我们可以列出如下平衡条件：

\begin{eqnarray} \alpha_{j H} \gamma_{j} Y_{j t} C_{j t}^{\theta-2} C_{t}^{-\theta} =\phi\left(H_{j t} / H_{t}\right)^{v} &&\quad{(\text{for all }j)}\notag\\ \gamma_{i} C_{i t}^{\theta-2} M_{i j t}^{1-\theta} =\gamma_{j} \alpha_{j M} \mu_{i j} Y_{j t} C_{j t}^{\theta-2} M_{j t}^{-\theta} &&\quad(\text{for all }i,j)\notag\\ C_{j t}^{\theta-2} C_{t}^{-\theta} =\beta \mathbb{E}_{t}\left[C_{j, t+1}^{\theta-2} C_{t+1}^{-\theta}\left(\alpha_{j K}\left(Y_{j, t+1} / K_{j t}\right)+1-\delta\right)\right] &&\quad{(\text{for all }j)}\notag\\ Y_{j t} =\left(e^{Z_{t}} H_{j t}\right)^{\alpha_{j H}} K_{j, t-1}^{\alpha_{j K}} M_{j t}^{\alpha_{j M}} &&\quad{(\text{for all }j)}\notag\\ Y_{j t} = C_{j t}+I_{j t}+\sum_{k} M_{j k t} &&\quad{(\text{for all }j)}\notag\\ Z_{t} =\rho Z_{t-1}+\varepsilon_{t} &&\notag\\ \varepsilon_{t} \sim \mathcal{N}\left(0, \sigma^{2}\right), \text { i.i.d. }&&\notag \end{eqnarray}

4. 结论

正如我们在引言中所指出的，本文的主要目的是证明DSGE模型可以用于分析经济结构，并扩展DSGE模型使其能解释现有模型没有捕捉到的一些现实生活现象。我们在这个方向上迈出了第一步，设计了一个简单的基于RBC的DSGE模型，该模型可以容纳任意数量的部门，这些部门之间的相互作用由国民账户按照公认的欧盟会计准则编制的投入产出表来描述。在第三章中，我们推导了可以输入到广泛使用的Dynare软件包中的平衡条件。

然而，即使在这种简单的设置中，在模型可以用于实际数据之前，仍有一些实际障碍需要解决。具体来说，需要有效的稳态数值解。到目前为止，我们只得到了一些初步的结果，即在命题1中描述的对称问题的特殊情况下的稳态特性。将来，我们的目标是测试这些结果在多大程度上可以用来为解决一般情况下问题的数值算法找到初始猜测。

未来研究的其他目标包括根据捷克数据校准模型，并将基线模型扩展到更精细的设置；这些扩展可能如下：

  • 更多行业特定参数（如折旧率、技术冲击等）
  • 引入政府和外国部门
  • 明确对待劳动力市场及其不完善之处
  • 纳入可能的价格、劳动力和资本调整成本
