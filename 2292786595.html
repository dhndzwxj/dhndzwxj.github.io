MWG第一章：偏好

|字数总计:2.5k|阅读时长:9分钟|阅读量:
图1 本章大纲

感谢知乎用户economics-lover。

首先，我们有一些备选物品集合（或者说商品集合）$X $ ，本质上是想建立关于消费者选择行为的模型，也就是说刻画消费者如何选择商品。有两种办法：第一章是基于偏好的办法；第二种是基于直接观察消费者选择行为的办法。这两种办法各有优劣，但第一种方法更为常用。

1. 基于偏好的消费者选择模型

偏好 $ \succsim $ 本质上是建立在被选集上的二元关系，如果 $x\gtrsim y $ ，那么我们说 $ x $ 至少和 $ y $ 一样好； $ x \succsim y$ 意味着 $ x $ 严格比 $ y $ 好； $ x \sim y$ 意味着 $ x $ 和 $ y $ 无差异。

通常，我们假设偏好是理性的，也就是说偏好满足如下两个性质：

  • 完备性：任何 $ x,y $ 两个均可比较， ∀x,y∈X\forall x,y \in X∀x,y∈X ，要么 $ x\succsim y $ ，要么 $ y\succsim x$ 。当然，二者可能同时成立——对应于无差异时的情形。
  • 传递性： $ \forall x,y,z\in X $ ，如果 $ x \succsim y,y\succsim z $ ，那么 $ x\succsim z$。

有以下命题成立（理性假设蕴含着严格偏好≻\succ≻和无差异∼\sim∼的性质）

命题1.B.1:如果 $ \succsim $ 是理性的，则：

  • ≻\succ≻为非反身（irreflexive）【x≻xx\succ xx≻x不成立】的和传递的（若$x\succ y,y\succ z $ ，则x≻zx\succ zx≻z）。
  • ∼\sim∼为反身的【x∼xx\sim xx∼x】和传递的（若$x\sim y,y\sim z $ ，则x∼zx\sim zx∼z）。

研究偏好往往是比较困难的，如果可以把偏好转化为效用函数，经常会使得分析简单很多，下面给出效用函数的定义（注意，效用函数是基于偏好的）：

一个效用函数u:X→Ru:X\rightarrow\mathbb{R}u:X→R表达了一个偏好$\succsim $ ，如果∀x,y∈X\forall x,y\in X∀x,y∈X：

x≿y⇔u(x)≥u(y)x\succsim y\Leftrightarrow u(x)\ge u(y) x≿y⇔u(x)≥u(y)

注意两点：

  1. 能表示某个偏好的效用函数不是唯一的，对于任何严格递增函数 f:R→R,v(x)=f(u(x))f:\mathbb{R}\rightarrow\mathbb{R},v(x)=f(u(x))f:R→R,v(x)=f(u(x))都是一个新的效用函数（你需要去证明这件事情）
  2. 不是任何偏好都可以找到效用函数来表达（你也需要去证明这件事情，比如一个二元的字典式偏好，不过现在不用着急，第二章还会讲到）

有以下命题成立：

命题1.B.2：如果一个偏好关系可以用效用函数表示，那么其一定是理性的——满足完备性和传递性

证明也比较简单，此处就略过。

好，这部分就把基于偏好的个体选择行为刻画清楚了，下一部分是基于选择规则的个体选择行为。

2. 基于选择规则的个体选择行为

下面整理基于选择规则的个体选择行为，选择行为用选择结构刻画，选择结构：(B,C(⋅))(\mathbb{B},C(\cdot))(B,C(⋅))。

  1. 其中B\mathbb{B}B是一个集簇，由XXX的非空子集构成，称B∈BB\in\mathbb{B}B∈B为预算集。
  2. $ C(\cdot)$ 是一个选择规则，满足 $ C(B)\subseteq B $ ，也就是说给定一个预算集，从预算集里面选择一个或多个元素。

例1.C.1
假设 $ X = { x, y, z }, \mathbb{B}= { { x , y } , { x,y,z } } $ ，一种可能的选择规则是 (B,C1(⋅))(\mathbb{B},C_ 1(\cdot))(B,C1​(⋅))：$ C_ 1({ x,y })={ x } $ ，C1({x,y,z})={x}C_ 1(\{ x,y,z \})=\{ x \}C1​({x,y,z})={x} 。也就是说，不管消费者面对什么样的预算集，都会选择 $ x $ 。这里可以隐约的感觉出来，似乎也刻画了偏好的某些东西，至少这里意味着 $ x \succ y,x\succ z $ ，但是我们并不知道 y,zy,zy,z 之间如何比较。

接下来，引入选择结构的弱公理，选择结构弱公理刻画了选择结构应该具备的合理性质：

定义1.C.1(显示偏好弱公理): 若选择结构(B,C(⋅))(\mathbb{B},C(\cdot))(B,C(⋅))具有下列性质：

若对于满足 x,y∈Bx,y\in Bx,y∈B 的 $B\in\mathbb{B} $ ，我们有 $x\in C(B) $ ，那么对于任何 $ x,y\in B’ $ 和 y∈C(B′)y\in C(B')y∈C(B′) 的任何 $y\in C(B’) $ ，我们必有 C(B′)C(B')C(B′) 。

那么我们就说选择结构 (B,C(⋅))(\mathbb{B},C(\cdot))(B,C(⋅)) 满足显示偏好弱公理。

这里的意思实际上是说：如果在某个选择过程中，我们发现个体同时可以选 $ x,y $ 的时候，个体选择了xxx而没有选择yyy。那么我们希望个体的这种选择具有一致性，就是在之后遇到同时可以选择x,yx,yx,y的时候，如果个体选择了 $ y $ ，那么一定要选 $ x $ 。

弱公理还有一种更简单的表示方法：

定义1.C.2：给定一个选择结构$(\mathbb{B},C(\cdot)) $ ，显示偏好关系≿⋆\succsim^\star≿⋆定义为：

x≿⋆y⇔∃B∈B,s.t. x,y∈B,x∈C(B)x\succsim^\star y\Leftrightarrow \exists B\in\mathbb{B},\text{s.t. }x,y\in B,x\in C(B) x≿⋆y⇔∃B∈B,s.t. x,y∈B,x∈C(B)

我们称x≿⋆yx\succsim^\star yx≿⋆y为：xxx被显示至少与yyy一样好。

如果上面的是：

x≿⋆y⇔∃B∈B,s.t. x,y∈B,x∈C(B),y∉Bx\succsim^\star y\Leftrightarrow \exists B\in\mathbb{B},\text{s.t. }x,y\in B,x\in C(B),y\notin B x≿⋆y⇔∃B∈B,s.t. x,y∈B,x∈C(B),y∈/​B

那么称xxx被显示比yyy更受偏好。

3. 两种刻画个体选择行为方式之间的关系

它们的关系可以用以下两个问题概括：

  • 给定某个理性的偏好关系，个体在预算集B\mathbb{B}B中做出的决策，必然能产生符合弱公理的选择结构吗？（理性偏好关系⇒\Rightarrow⇒选择结构）
  • 如果个体在预算集B\mathbb{B}B中做出的决策符合弱公理，一定可以找到与这些选择相符的理性偏好关系吗？（选择结构⇒\Rightarrow⇒理性偏好关系）

第一个问题的答案是肯定的：假设个体在XXX上具有理性偏好，当面对某个子集B⊆BB\subseteq \mathbb{B}B⊆B时，消费者的偏好最大化行为是在其中选择一个最好的元素，即：

C⋆(B,≿)={x∈B:x≿y,∀y∈B}C^\star(B,\succsim)=\{ x\in B: x\succsim y,\forall y\in B \} C⋆(B,≿)={x∈B:x≿y,∀y∈B}

在一些微弱的条件下，C⋆(B,≿)C^\star(B,\succsim)C⋆(B,≿)一定是非空的，此时我们说理性偏好关系生成了选择结构： $(\mathbb{B},C^\star(\cdot,\succsim)) $ ，为什么这么说呢？因为一个选择结构，最重要的就是选择规则，这里的选择规则是依赖于这个理性偏好的，因此我们说该理性偏好关系生成了选择规则。

命题1.D.1：假设≿\succsim≿是一个理性偏好关系，则由≿\succsim≿生成的选择结构$$(\mathbb{B},C(\cdot))$$满足弱公理。

为了回答第二个问题，引入一个新的概念：

定义1.D.1:给定一个选择结构$(\mathbb{B},C(\cdot)) $ ，一个偏好关系理性化了该选择结构，如果∀B∈B\forall B\in\mathbb{B}∀B∈B：

C(B)=C⋆(B,≿)C(B)=C^\star(B,\succsim) C(B)=C⋆(B,≿)

也就是说理性偏好生成了选择结构(B,C(⋅))(\mathbb{B},C(\cdot))(B,C(⋅))。

上述命题告诉我们这么一件事情，一个选择结构，只有满足了弱公理，才能被理性化。因为，理性偏好生成的选择结构一定满足弱公理，那么如果一个选择结构不满足弱公理，那么一定不能被理性化，因为如果不满足弱公理，那么(B,C(⋅))(\mathbb{B},C(\cdot))(B,C(⋅))不可能由理性偏好生成，即C(B)≠C⋆(B,≿),∀B∈BC(B)\neq C^\star(B,\succsim),\forall B\in\mathbb{B}C(B)​=C⋆(B,≿),∀B∈B不可能成立。

尽管一个选择结构，只有满足了弱公理，才能被理性化，但反之则不成立。下面是一个反例：

例1.D.1：假设X={x,y,z}X=\{ x,y,z \}X={x,y,z},B={{x,y},{y,z},{z,x}}\mathbb{B}=\{ \{ x,y \},\{ y,z \},\{ z,x \} \}B={{x,y},{y,z},{z,x}},C({x,y})={x},C({y,z})={y},C({z,x})={z}C(\{ x,y \})=\{ x \},C(\{ y,z \})=\{ y \},C(\{ z,x \})=\{ z \}C({x,y})={x},C({y,z})={y},C({z,x})={z}。

容易验证，该选择结构满足弱公理，但是该选择结构不能被理性化。

为什么选择结构可以表示比偏好更多的东西呢？因为我们看到偏好生成的选择结构C⋆(B,≿)={x∈B:x≿y,∀y∈B}C^\star(B,\succsim)=\{x\in B:x\succsim y,\forall y\in B\}C⋆(B,≿)={x∈B:x≿y,∀y∈B}存在局限性，它只能对的单个元素进行考虑，却没有办法深入考虑互补和替代，比如牛奶比可乐还有薯条都要好，但是薯条+可乐的组合比牛奶要好，偏好关系生成的选择结构没办法很好地刻画这个东西。

不过，如果选择结构在满足弱公理的条件上再增加一些条件——预算集中包含足够多的子集，那么存在可以理性化选择结构(B,C(⋅))(\mathbb{B},C(\cdot))(B,C(⋅))的理性偏好关系。

定义1.D.1:给定一个选择结构$(\mathbb{B},C(\cdot)) $ ，如果满足

  • 满足弱公理
  • B\mathbb{B}B包含了三元及三元以下的所有子集

则存在能理性化(B,C(⋅))(\mathbb{B},C(\cdot))(B,C(⋅))的理性偏好关系$\succsim $ ，也就是说， $\forall B\in\mathbb{B},C(B)=C^\star(B,\succsim) $ ，并且这样的理性偏好是惟一的。
