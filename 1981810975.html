rucbeamer模板

发表于2021-03-22|更新于2021-07-26|杂技LaTeX
|字数总计:1.4k|阅读时长:5分钟|阅读量:

改好的ruc_beamer模板，亲测可用

\documentclass[UTF8,aspectratio=43]{ctexbeamer}
%aspectratio是长宽比，有1610，169，43，54这些参数
\usepackage{ruc}
\usepackage{graphicx}
\usepackage{subfigure}
\usepackage{setspace}
\usepackage{ragged2e}
\usepackage{indentfirst}


% 字体主题
\usefonttheme{serif}
\setmainfont{Times New Roman}%主要的英文字体
% \setCJKmainfont{heiti}%主要的中文字体
% 自定义字体名称，目前可用的只有heiti,kaishu,fangsong,simsun
% \newcommand{\xingkai}{\CJKfontspec{STXingkai}}%华文行楷
% 各组成部分字体设置
\setbeamerfont{title}{size=\Huge}	%标题字体
\setbeamerfont{subtitle}{size=\Large}	%副标题字体
\setbeamerfont{author}{family=\kaishu}	%作者字体
\setbeamerfont{section in toc}{family=\kaishu}	%目录字体
\setbeamerfont{date}{family=\heiti}	%日期字体
\setbeamerfont{itemize/enumerate body}{size=\large,family=\kaishu} %列举条目字体
\setbeamerfont{itemize/enumerate subbody}{size=\normalsize,family=\fangsong} %列举二级条目字体
% \setbeamerfont{block title}{family=\heiti}

\begin{document}
\setbeamertemplate{footline}[frame number] %页脚

\begin{frame}

	\begin{itemize}
		\item 半个世纪以来，制造业产业的生产过程在全球范围内逐步展开，分工在国际范围内越来越精细化，多国都融入了全球生产链条和全球市场中，在某生产环节上起到至关重要的作用，逐步形成了国际生产分割（international production segmentation）的现象
		\begin{itemize}
			\item[-] 全球化生产分割实例：iPhone12系列
			\item[-] 区域化生产分割实例：液晶面板\medskip
		\end{itemize}
		\item 国际生产分割是更加区域化还是全球化？\medskip
		\item 本文尝试回答这个问题	\medskip
	\end{itemize}
\end{frame}
