windows注册表

揭晓|发表于2023-03-06|更新于2023-03-14|杂技Windows终端

添加右键菜单项

  1. 右键单击 shell，新建（N） -> 项（K） ，创建一个新项（名字最好是功能相关）；
  2. 单击新项，然后 双击 右侧 （默认） 值，写入 数据 内容（右键菜单项显示的文本）；
  3. 右键单击第一步创建的新项，创建一个子项，命名为 command；
  4. 单击 command ，双击右侧 （默认） 值，填写你要执行的命令；

各项菜单的注册表路径

桌面空白处：计算机\HKEY_CLASSES_ROOT\DesktopBackground\Shell

文件：计算机\HKEY_CLASSES_ROOT\*\shell

文件夹：计算机\HKEY_CLASSES_ROOT\Directory\shell

文件和文件夹：计算机\HKEY_CLASSES_ROOT\AllFilesystemObjects\shell

文件夹空白处：计算机\HKEY_CLASSES_ROOT\Directory\Background\shell

应用：在ConEmu中打开右键单击菜单窗口

设置功能集成(或旧版本中的设置→功能→集成)

设置ConEmu Here组(不要在内部滥用ConEmu )，然后按注册按钮。

Windows11恢复Windows10右键菜单样式

切换到旧版右键菜单：

taskkill /f /im explorer.exe
start explorer.exe

报错的话按上面分两行执行。
