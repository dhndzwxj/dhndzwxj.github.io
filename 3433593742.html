什么是ipv6？

揭晓|发表于2022-11-30|更新于2023-03-07|杂技网络
|字数总计:576|阅读时长:2分钟|阅读量:

NAT：网络地址转换

现在ip地址不够用，只能重复使用。这种情况在私人网络（局域网）的环境下并不会出多大问题，但一旦要接入互联网，就会出现ip地址重复的问题。这时候需要路由器将私网ip转换为公网ip。

NAT的意义是，在一个局域网下，很多人共用一个ip（192.168.X.X）。但这增加了设备的压力，降低了设备性能。【ip转换，这也是因为移动干不过电信，因为移动没有公网ip】

IPV6

IPV6三大优势在于：

  1. ipv6的ip地址数量多。 $ 2^{128} $ 个。
  2. ipv6性能更好
  • 可以不做nat，直接出去
  • ipv6简化报头，网络设备处理起来更加简洁
  • ipv6取消了广播，减少了无用信息的干扰
  1. ipv6更加安全

IPV6地址缩写

ipv6地址总长度为128bit，分为8组，每组为4个十六进制数的形式，每组用冒号来分隔。

十六进制数：123456789ABCDEF

IPV6地址例子：FC00:0000:130F:0000:0000:09C0:876A:130B

简写方法：

  • 每一组前面的“0”可以省略：FC00:0:130F:0:0:9C0:876A:130B
  • 地址中包含的连续两个或者多个0的组，可以用双冒号::来代替，于是上述地址又可以进一步简化为：FC00:0:130F::9C0:876A:130B

例子：一个合法的ipv6地址：2002::1=2002:0000:0000:0000:0000:0000:0000:0001

配置IPV6地址【华为设备为例】

ipv6 #回车，代表给设备开启ipv6功能
int g0/0/0 #进入接口
ipv6 enable #给接口开启ipv6功能
ipv6 add 2022::1

Error:Incomplete commmand found at '^' position. #不完整

ipv6 add 2022::1 64

ipv6 on the interface ^ has entered the UP state.

#ipv4地址配置
ip add 192.168.1.1 24 #24是掩码

掩码是为了确定主机位。区分网段，是否找网关。

网络位的作用：判断两个主机是否在同网段。

例子：

192.168.1.1 255.255.255.0

192.168.2.1 255.255.255.0

这个掩码说明二者不在同一个网段

192.168.1.1 255.255.0.0

192.168.2.1 255.255.0.0

这个掩码说明二者在同一个网段

2000:2000:2000:0000::1 64

2000:2000:3000:0000::1 64

这个掩码说明二者不在同一个网段
