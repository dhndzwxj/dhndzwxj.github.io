王勇垂直一体化模型

揭晓|发表于2021-12-16|更新于2023-01-05|计量与模型模型
|字数总计:883|阅读时长:3分钟|阅读量:

李系,刘学文,王勇.一个中国经济发展的模型[J].经济学报,2014,1(04):1-48.

DOI:10.16513/j.cnki.cje.2014.04.001.

观点：

  • 2001-2011年间，国企展现出比非国企更高的整体盈利能力——2001年，国企从下游开放产业退出，非国有在自由竞争的下游快速扩张会增加对上游中间品和服务的需求。如果没有国际贸易的外来需求、没有充足的劳动力（工资上涨），上游国企也无法持续获得高利润
  • （同一时间段，北方发展也略好于南方。自然可以讲一讲南北分工形成的历史、上下游产业特征作为垄断与竞争的解释——同时也有北方下游企业竞争力不足，逐渐退出，消费品在南方的兴起）
  • 国有企业垄断关键的上游产业，下游产业基本上对民企开放并允许自由竞争
  • 上游的国有企业在结构转型和全球化过程中，从开放的下游产业中获得垄断租金

描述性统计、机制检验

假设有工农两类工业部门，模型考察只考察了非农部门的垂直结构（中间-最终），本文只强调中间产品市场的垄断扭曲。

3 基准模型

三类产品：农业品 $ n $ 、综合消费品 $ d $ 、中间品 $ m $ ；前二者为数量 $ 1-\theta $ 的草根拥有，竞争性市场；后者为数量 $ \theta $ 精英拥有，垄断市场。

3.1 偏好与生产函数

所有个体（草根和精英）有相同的效用函数：

\begin{eqnarray} u( c_n , c_d ) = c_n + \frac{\varepsilon}{\varepsilon-1} c_d ^{\frac{\varepsilon-1}{\varepsilon}}, \quad \varepsilon>1 \tag{1} \end{eqnarray}

其中 $ \varepsilon $ 是商品 $ d $ 的需求价格弹性。

由于本文只关注工业部门，因此只给出了上游和下游两类产品的生产函数：

\begin{eqnarray} && F_d (K,L,M) = AK^\alpha L^\beta M^{1-\alpha-\beta},\quad 0<\alpha,\beta<1,\alpha+\beta<1 \tag{2}\\ && F_m (K,L) = A_m K^\gamma L^{1-\gamma}, \quad 0<\gamma<1 \tag{3} \end{eqnarray}

发现综合消费品生产投入除了资本和劳动外，还有中间品。从后文我们可以推断，农业部门的生产函数应为：

\begin{eqnarray} F_n = A_n L \notag \end{eqnarray}

3.2 禀赋与市场结构

每个个体都有 $ L $ 单 位时间和 $ K $ 单位资本。二者的收入分别为：

\begin{eqnarray} && I_g = WL+RK \notag\\ && I_e = I_g + \Pi_m \notag \end{eqnarray}

因而其预算约束写为：

\begin{eqnarray} p_n c_n +p_d c_d \le I, \quad I \in \{ I_g, I_c \} \notag \end{eqnarray}

3.3 消费者需求求解

求解对两种产品的需求函数：

\begin{eqnarray} && u = c_n + \frac{\varepsilon}{\varepsilon-1} c_d ^{\frac{\varepsilon-1}{\varepsilon}}, \quad \varepsilon>1 \notag\\ \text{s.t.} && p_n c_n + p_d c_d = WL + RK + \Pi_m \notag \end{eqnarray}

构造拉格朗日函数：

\begin{eqnarray} \mathscr{L} = c_n + \frac{\varepsilon}{\varepsilon-1} c_d ^{\frac{\varepsilon-1}{\varepsilon}}+\lambda (WL+RK+ \Pi_m - p_n c_n - p_d c_d) \notag \end{eqnarray}

求其一阶条件，很容易解出：

\left\{\begin{eqnarray} && D_n = \frac{WL+RK+\Pi_m}{p_n} - \left( \frac{p_n}{p_d} \right)^{\varepsilon-1} \tag{4}\\ && D_d = \left( \frac{p_n}{p_d} \right)^{\varepsilon} \tag{5} \end{eqnarray}\right.

3.4 竞争性下游企业决策求解

\begin{eqnarray} \max_{ \textbf{z} } \quad p_d F_d(\textbf{z}) - \textbf{w}^T \textbf{z} = p_d A K^\alpha L^\beta M^{1-\alpha-\beta} - (RK+WL+ p_m M) \notag \end{eqnarray}

其一阶条件写成向量形式：

\begin{eqnarray} \nabla_{\textbf{z}} F_d(\textbf{z}) = \frac 1{p_d} \textbf{z} \notag \end{eqnarray}

方程组形式：

\begin{eqnarray} && A\alpha K^{\alpha-1} L^\beta M^{1-\alpha-\beta} = \frac{R}{p_d} \tag{①}\\ && A\beta K^{\alpha} L^{\beta-1} M^{1-\alpha-\beta} = \frac{W}{p_d} \tag{②}\\ && A(1-\alpha-\beta) K^{\alpha} L^\beta M^{-\alpha-\beta} = \frac{p_m}{p_d} \tag{③} \end{eqnarray}

①②消去 $ M $ : $ L = \frac{\beta KR}{\alpha W} $

①③消去 $ L $ ： $ M = \frac{(1-\alpha-\beta)KR}{\alpha p_m} $

这两个式子有个规律，对于不同的要素，其数量乘以其价格并除以其成本份额，结果值是一样的！

再代入第一个式子里：

\begin{eqnarray} A \alpha k^{\alpha-1} \left( \frac{\beta KR}{\alpha W} \right)^\beta \left[ \frac{(1-\alpha-\beta) KR}{\alpha p_m} \right]^{1-\alpha-\beta} = \frac{R}{p_d} \tag{④} \end{eqnarray}

式子④经过演算，发现式子中的 $ K $ 的被消去了，最后得到了综合消费品的定价：

\begin{eqnarray} p_d = \frac{R^\alpha W^\beta p_m^{1-\alpha-\beta}}{A\alpha^\alpha \beta^\beta (1-\alpha-\beta)^{1-\alpha-\beta} } \tag{6} \end{eqnarray}

3.5 垄断上游厂商的决策求解

下面一步要算出来下游厂商对中间品 $ M $ 的需求量有多大，问题转化为给定要素价格 $ R, W ,p_m $ 和产量 $ D_d $ 的前提下，计算中间品 $ M $ 的需求量：

\begin{eqnarray} && \min_{K,L,M} \quad C=RK+WL+ p_m M \notag\\ \text{s.t.} && A K^\alpha L^\beta M^{1-\alpha-\beta} = D_d = \left( \frac{p_n}{p_d} \right)^\varepsilon \notag \end{eqnarray}

借用上面的”规律“，用 $ M $ 表示 $ K $ 和 $ L $ ：

\begin{eqnarray} && K = \frac{\alpha p_m M}{(1-\alpha-\beta) R} \notag\\ && L = \frac{\beta p_m M}{(1-\alpha-\beta) W} \notag \end{eqnarray}

代入上面的约束条件中：

\begin{eqnarray} A \notag \left[ \frac{\alpha p_m M}{(1-\alpha-\beta) R} \right]^\alpha \left[ \frac{\beta p_m M}{(1-\alpha-\beta) W} \right]^\beta M^{1-\alpha-\beta} = \left( \frac{1}{p_n}\cdot \frac{R^\alpha W^\beta p_m^{1-\alpha-\beta}}{A\alpha^\alpha \beta^\beta (1-\alpha-\beta)^{1-\alpha-\beta} } \right)^{-\varepsilon} \notag \end{eqnarray} \begin{eqnarray} \Rightarrow A \frac{\alpha^\alpha \beta^\beta}{R^\alpha W^\beta} \cdot \left(\frac{p_m}{1-\alpha-\beta}\right)^{\alpha+\beta}\cdot M = \left( \frac{1}{p_n}\cdot \frac{R^\alpha W^\beta p_m^{1-\alpha-\beta}}{A\alpha^\alpha \beta^\beta (1-\alpha-\beta)^{1-\alpha-\beta} } \right)^{-\varepsilon} \notag \end{eqnarray}

解出来：

\begin{eqnarray} D_m \overset{\Delta}{=} M = (1-\alpha-\beta) p_n^{\color{red} \varepsilon } \left[ \frac{R^\alpha W^\beta}{A\alpha^\alpha \beta^\beta (1-\alpha-\beta)^{1-\alpha-\beta}} \right]^{1-\varepsilon} p_m^{(1-\alpha-\beta)(1-\varepsilon)-1} \tag{7} \end{eqnarray}

为了下文的计算方便，可以简单记为：

\begin{eqnarray} D_m =D_0\cdot p_m^{(1-\alpha-\beta)(1-\varepsilon)-1} \notag \end{eqnarray}

根据3.4的竞争性情形，上游企业的竞争性定价应为：

\begin{eqnarray} p_m^c = \frac{R^\gamma W^{1-\gamma}}{A_m \gamma^\gamma (1-\gamma)^{1-\gamma}} \notag \end{eqnarray}

于是垄断上游企业的最优决策问题转换为：

\begin{eqnarray} \Pi_m &=& \max_{p_m} \quad D_m \left[ p_m- p_m^c \right] \tag{8}\\ &=& \max_{p_m} \quad D_0\cdot p_m^{(1-\alpha-\beta)(1-\varepsilon)-1}\left[ p_m- p_m^c \right] \notag\\ &=& \max_{p_m} \quad D_0 \left[ p_m^{(1-\alpha-\beta)(1-\varepsilon)} - p_m^c \cdot p_m^{(1-\alpha-\beta)(1-\varepsilon)-1} \right] \notag \end{eqnarray}

一阶条件等于0：

\begin{eqnarray} \frac{\partial \Pi_m}{\partial p_m} = D_0 \left[ (1-\alpha-\beta)(1-\varepsilon) p_m^{(1-\alpha-\beta)(1-\varepsilon)-1} - p_m^c p_m^{(1-\alpha-\beta)(1-\varepsilon)-2}\right] =0 \notag \end{eqnarray}

解出来：

\begin{eqnarray} p_m = \frac{(1-\alpha-\beta)(\varepsilon-1)+1}{(1-\alpha-\beta)(\varepsilon-1)} \cdot p_m^c \overset{\Delta}{=}\mu p_m^c \tag{9} \end{eqnarray}

这里的 $ \mu $ 是价格加成（常见诸后凯文献）,根据参数设定有 $ \mu>1 $ ，它由中间商品在下游商品生产中所占的成本份额 $ 1-\alpha-\beta $ 和对商品 $ d $ 的需求价格弹性 $ \varepsilon-1 $ 决定。

3.6 市场出清条件

根据谢泼德引理，在给定产量和要素价格的情况下，对要素的需求量为：

\begin{eqnarray} \textbf{z}(\textbf{w},\bar q) = \nabla_{\textbf{w}} C(\textbf{w},\bar q) \notag \end{eqnarray}

这里要算对劳动力的需求，由于产量是常数，因此可以作如下变换：

\begin{eqnarray} L = \frac{\partial C}{\partial W} = \frac{\bar q \cdot \partial p}{\partial W} \notag \end{eqnarray}

于是有劳动力市场出清条件：

\begin{eqnarray} L &=& \underbrace{D_m \frac{\partial p_m^c}{\partial W}}_{\text{中间产品生产对劳动力的需求}} + \underbrace{D_d \frac{\partial p_d}{\partial W}}_{\text{下游产品生产对劳动力的需求}} + \underbrace{D_n \frac 1 {A_n}}_{\text{农产品生产对劳动力的需求}} \tag{11}\\ &\overset{\Delta}{=}& \bar L + D_n \frac 1 {A_n} \notag \end{eqnarray}

同理，资本市场出清条件：

\begin{eqnarray} K = \underbrace{D_m\frac{\partial p_m^c}{\partial R}}_{\text{中间品生产对资本的需求}} + \underbrace{D_d\frac{\partial p_d}{\partial R}}_{下游品生产对资本的需求} \notag \end{eqnarray}
